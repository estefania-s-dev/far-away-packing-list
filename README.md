# ✈️ Far Away - Travel Packing List

> **Status:** A functional React application for managing travel essentials, featuring item categorization, sorting, and packing statistics.

This project was built to practice modern React patterns, specifically **State Management**, **Lifting State Up**, and **Derived State**. It allows users to plan their trips by adding items, marking them as packed, and viewing a real-time "completion" percentage.

---

## 🚀 Key Features

- **Dynamic Item Management:** Add, delete, and toggle "packed" status for travel items.
- **Sorting Logic:** Sort items alphabetically, by input order, or by packed status.
- **Real-time Statistics:** A footer that calculates the total number of items and the percentage of items already packed.
- **Confirmation Guards:** Prevents accidental deletion of the entire list.

---

## 🛠️ Technologies Used

- **Framework:** React.js
- **State Handling:** `useState` hook for list management.
- **Logic:** Array methods (`map`, `filter`, `sort`, `reduce`) for derived state.
